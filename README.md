# DSA Series

This repository is a C++ data structures and algorithms learning collection organized as a series.

## What’s inside

- `LectureXXX ...` folders: C++ examples and practice problems covering
  - programming basics, control flow, loops, and pattern printing
  - arrays, pointers, functions, and dynamic memory
  - recursion, sorting, searching, and number theory
  - object-oriented programming, linked lists, stacks, queues, and trees
- `Graph/`: graph implementation examples and related algorithms
- `Heaps/`: heap operations, heapify, insertion, deletion, and heap sort

## How to use

- Open the project in VS Code, then open a lecture folder or example file.
- Use the existing task `C/C++: gcc.exe build active file` to compile the current `.cpp` file.
- Or compile manually:

```bash
g++ path/to/file.cpp -o path/to/file.exe
./path/to/file.exe
```

## Notes

- The repository contains both source `.cpp` files and some compiled `.exe` artifacts.
- Remove generated binaries before sharing the repository if you want a clean source-only version.

## Goal

This collection is intended for learning and practicing common DSA patterns in C++.
